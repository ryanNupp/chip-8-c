# chip-8-c
A chip 8 emulator written in C 

This project was planned to be written in C and use ncurses to work the display so this program could run all within the terminal.

Unfortunately, I ran into some issues when it came to input so I shelved the project for a bit.

This version never got finished, but I gave another shot at this using SDL3 and C++ which can be found [here](https://github.com/ryanNupp/chip-8-cpp).
